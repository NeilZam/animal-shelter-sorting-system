# Animal Shelter Sorting System

A sorting application written in C that ranks animals using multiple sorting algorithms and comparison strategies.

## Features

- Merge Sort
- Quick Sort
- Insertion Sort
- Hybrid sorting implementation
- Multi-key comparisons
- Dynamic memory allocation

## Concepts Used

- Divide and Conquer Algorithms
- Sorting Algorithms
- Recursion
- Runtime Analysis

## Technologies

- C
- GCC
- Command Line

## What I Learned

This project improved my understanding of sorting algorithm performance, recursion, and algorithmic efficiency.
